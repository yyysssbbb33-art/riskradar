# RiskRadar v0.4.1 → v0.4.2 마이그레이션

v0.4.2는 새로운 판정 모델을 추가하는 버전이 아니라, 이미 있는 **6개 상세 해석 카드와 현재 데이터를 직접 연결하는 UI 패치**다.

## 핵심 변경

`현재 상황` 탭의 표 아래에 핵심 6개 지표별 아코디언을 추가했다.

각 아코디언은 다음 순서로 표시한다.

1. **현재 데이터와 연결해서 보면**
   - 상태
   - 최신값
   - 관측일
   - 약 1개월 변화
   - 약 3개월 변화
   - 최근 5년 위치
   - 최근 10년 위치
   - 현재 지표의 한줄 해석

2. **현재 연결된 조합**
   - 오늘의 해석 엔진에서 이 지표와 직접 연결된 조합
   - 현재 상대적으로 더 잘 맞는 설명
   - 반대 증거·약해지는 설명
   - 결과가 엇갈리는 부분

3. **기존 8칸 상세 설명**
   - `interpretation_cards.py`의 원문을 그대로 재사용

## 구현 원칙

- 설명을 UI마다 복사하지 않는다.
- 기존 8칸 카드 원문은 `interpretation_cards.py` 한 곳에서만 관리한다.
- 현재 조합 연결은 조합 라벨 문자열 검색이 아니라 `combo_id → 핵심지표` 명시적 매핑으로 관리한다.
- 새 위험점수, 새 상태판정, 새 투자행동 문장은 추가하지 않았다.
- 경제일정·이벤트 기능은 추가하지 않았다.
- GitHub workflow 파일은 수정하지 않았다.
