# RiskRadar v0.7.0 표현 정리 패치

## 성격

이 패치는 v0.7.0의 기능·판정 규칙·데이터 수집·workflow를 바꾸지 않는 표시 문구 패치입니다.

## 주요 변경

- 지표 이름을 `쉬운 한국어(실제 약칭)`으로 통일했습니다.
  - 예: `주식시장 예상 변동성(VIX)`, `신용등급이 낮은 기업의 추가 금리(HY OAS)`
- `새로 상승`은 `상승 확인`으로 바꿨습니다.
- `최근 갱신에서 달라진 것`은 `새로 달라진 점`으로 바꿨습니다.
- `아직 남아 있는 것`은 `계속 이어지는 변화`로 바꿨습니다.
- `참여/미참여/에피소드/렌즈/잔존 변화` 같은 내부 엔진 표현은 사용자 화면에서 자연스러운 시장 표현으로 바꿨습니다.
- Telegram, 지표 설명, 관계 가이드, 지난 30일 흐름, 함께 볼 지표 카드의 용어도 같은 기준으로 맞췄습니다.

## 배포

일반 v0.7.0과 동일하게 배포하면 됩니다. 데이터 refresh는 필수가 아닙니다.
