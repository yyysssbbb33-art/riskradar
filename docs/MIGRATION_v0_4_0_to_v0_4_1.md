# RiskRadar v0.4.0 → v0.4.1 마이그레이션

v0.4.1은 Claude가 만든 v0.4.0 코어를 유지하면서, 빠졌던 사용자 친화 UI와 해석 계층을 확장한 버전이다.
경제일정·이벤트 플레이북은 이번 버전에서 제외했다.

## 유지한 v0.4.0 코어

- 보조지표 3개 수집
- 3축 복합 조망
- VIX 빠른 센서 / HY OAS 확인 센서 구조
- 조건부 해석 엔진
- 오늘의 해석 탭
- 보조 계층 실패가 핵심 6개를 깨뜨리지 않는 장애 격리

## v0.4.1 추가·복구

### 사용자 화면

- 영어 탭명을 한국어로 변경
- `20obs`, `60obs`, `5Y%`, `10Y%` 사용자 노출 제거
- 현재 상황 표에 한줄 해석 추가
- 내부 코드명 대신 지표명 표시
- Telegram 표현 한국어화

### 개별 지표 해석

6개 전부 8칸 상세 카드 제공:

- VIX
- HY OAS
- 10년-3개월 금리차
- 미국 30년물 금리
- 미국 2년물 금리
- 10년 실질금리

### 복합 해석

- 조합 4개 → 15개
- 강한 축 상태와 별개로 자기 역사 대비 방향을 계산해 일반적인 금리 엇갈림도 탐지
- 조합 영역별 우선순위와 중복 제거
- `현재 더 잘 맞는 설명`
- `반대 증거·약해지는 설명`
- `결과 충돌`
- `아직 결정하기 어려운 부분`

### VIX 보완

지속성만 보던 기존 규칙에 자기 역사 대비 약 1개월 변화폭 필터를 추가했다.

- 즉시 강한 상태는 바로 활성
- 일반 관찰 상태는 최근 지속성 + 자기 역사 대비 변화 크기 확인
- 컷은 C등급 운영 규칙

### 데이터 품질

- stale 보조지표는 가설 지지·약화 집계에서 제외
- delayed 자료는 지연 표시
- 같은 날짜 여러 refresh는 원본 버전에는 보존하되 최근 30일 UI에는 마지막 스냅샷 1개만 표시

## 테스트

- 전체 72개 통과
- `pytest -q` 직접 실행 가능
- 사용자 화면 기술용어 회귀 테스트 포함
- VIX 자기 역사 필터 테스트 포함
- stale 보조지표 제외 테스트 포함

## 변경하지 않은 부분

사용자 요청에 따라 아래 워크플로 파일은 건드리지 않았다.

- `.github/workflows/refresh.yml`
- `.github/workflows/sync-to-hf-space.yml` (실제 repo에 존재하는 경우)

경제일정·FOMC·고용·CPI 이벤트 플레이북도 이번 버전에는 넣지 않았다.
