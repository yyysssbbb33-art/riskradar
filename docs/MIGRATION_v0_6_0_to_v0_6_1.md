# RiskRadar v0.6.0 → v0.6.1

v0.6.1은 판정 로직과 데이터 산출물을 바꾸지 않는 UI 정리 버전입니다.

## 핵심 변경

- `지표 설명`을 제외한 모든 탭에서 핵심 결과를 먼저 노출합니다.
- 읽는 법, 용어, 상세 근거, 대안 해석, 데이터 구성 방식, 운영 세부는 기본적으로 닫힌 아코디언 안으로 이동합니다.
- `오늘의 해석`은 핵심 요약을 새로 만들고 기존 전체 해석을 아코디언에 그대로 보존합니다.
- `현재 상황`과 `전체 지표 비교`의 첫 표는 핵심 열만 남겼습니다. 설명성 정보는 삭제하지 않고 상세 아코디언으로 옮겼습니다.

## 데이터·배치 영향

- refresh 산출물 변경 없음
- 캐시 스키마 변경 없음
- workflow 변경 없음
- v0.6.0 데이터와 그대로 호환
- v0.6.1 화면이 v0.6.0 데이터 캐시를 읽을 때는 정상 호환으로 처리하며 버전 불일치 경고를 띄우지 않음
