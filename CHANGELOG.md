## 0.6.0

- 기업 신용 분석을 `순서 엔진`이 아니라 **범위·지속 엔진**으로 재설계. 1차 산출물은 선행/후행이 아니라 현재 에피소드, 참여 시장, 노드 상태, 잔존 변화, 저신용 집중도다.
- 실제 시장 노드를 `HY / BBB / A / CP`로 구성. `HY-BBB`는 별도 시장이 아닌 **등급 차별 해석 렌즈**로 분리해 같은 움직임의 이중계상을 방지.
- broad IG(`BAMLC0A0CM`)는 BBB를 포함하므로 독립 확산 노드와 일반 상세 목록에서 제외. 구버전 캐시 호환과 운영 진단을 위해 수집 경로만 유지.
- 신용 노드 상태를 `평소 상태 / 초기 변화 / 새로 상승 / 상승 지속 / 되돌림 / 정상화`로 구성. `초기 변화`는 공식 참여 범위에 세지 않음.
- 에피소드 상태를 `활성 / 휴면 / 종료`로 분리. 최근 의미 있는 활동이 없지만 잔존 변화가 남으면 휴면 처리하고, 휴면 뒤 새 변화는 새 에피소드로 시작하면서 이전 잔존 변화를 주석으로 보존.
- 변화 시작 판정과 잔존 판정을 분리. 변화 강도로 후보·확정을 잡고, 확정 시 동결한 사전 기준선 대비 잔존 변화로 지속·되돌림·정상화를 판정.
- 확정 시 `추정 시작일 / 확정일 / 사전 기준선`을 동결해 이후 소급 수정하지 않음. 후보 시작일 직전 유효 관측치 중앙값을 기준선으로 사용해 초기 상승분 혼입 방지.
- `candidate_start / estimated_onset / confirmed_at / peak_at / last_meaningful_activity_at / normalized_at`을 v1부터 저장하되 사용자 출력에서는 선행·후행을 주장하지 않음.
- VIX를 신용 사슬 노드가 아닌 `에피소드 시작 무렵 주식시장 불안도 있었는가`를 보는 맥락 정보로 추가. NFCI·STLFSI는 계속 외부 후속 참고로만 유지.
- 기존 `주식시장이 먼저 움직임` 표현을 `주식시장 쪽만 움직임`으로 변경. 지난 30일의 `순서` 표현을 `새 변화가 처음 확인된 날짜`로 바꾸고 선행·후행 판정이 아님을 명시.
- FRED의 ICE BofA 회사채 계열 최근 약 3년 자료 정책에 맞춰 HY 위치를 `최근 3년`으로 전환. 실제 달력 커버리지가 부족하면 5년·10년 백분위를 계산하지 않아 3년 자료를 5년 위치로 오표시하던 문제 수정.
- 회사채 최근 3년 자료를 장기 역사 희귀도나 과거 위기 수준으로 부르지 않음. BBB·A 상세 화면은 `현재 확보된 자료 범위 안 위치`로 표시.
- 확인 지표를 `BREAKEVEN / TERMPREM / BBB / A / CP` 5개로 재구성하고 NFCI·STLFSI 2개를 별도 외부 참고층으로 분리. 사용자 상세 가이드는 핵심 6 + HY-BBB 렌즈 + 확인 5 + 외부 참고 2 = 14개.
