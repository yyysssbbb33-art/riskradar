# RiskRadar v0.4.2 — 미국 매크로 스트레스 계기판

미국 시장의 변동성·신용·경기 사이클·금리를 **따로 보고, 함께 읽는** 읽기 전용 대시보드입니다.
단일 위험점수, 매수·매도 신호, 침체 확률을 만들지 않습니다.

## 핵심 구조

```text
핵심 6개 지표
VIX · HY OAS · 10년-3개월 금리차 · 30년물 · 2년물 · 10년 실질금리

        ↓

3축 복합 조망
변동성·신용 / 경기 사이클 / 금리 방향

        ↓

조건부 해석
관찰된 조합 → 가능한 설명 → 보조지표 확인 → 지지·약화 → 충돌·불확실성
```

보조지표 3개는 원인 확인에만 사용합니다.

- 10년 기대인플레이션 (`T10YIE`)
- 투자등급 신용스프레드 (`BAMLC0A0CM`)
- 10년 Term Premium, Kim-Wright (`THREEFYTP10`)

## v0.4.2 주요 변경

- 사용자 화면의 영어 탭명을 한국어로 통일
- `20obs / 60obs / 5Y% / 10Y%` 노출 제거
  - 약 1개월 변화
  - 약 3개월 변화
  - 최근 5년 위치
  - 최근 10년 위치
- 핵심 6개 지표 모두 8칸 상세 해석 카드 제공
- 현재 상황 표에 `한줄 해석` 추가
- 현재 상황에서 6개 핵심 지표를 각각 펼쳐 `현재 데이터 연결 해석 + 현재 조합 + 8칸 상세 설명`을 바로 확인
- 오늘의 해석에 다음 구조 추가
  - 현재 상대적으로 더 잘 맞는 설명
  - 반대 증거·약해지는 설명
  - 결과 충돌
  - 아직 결정하기 어려운 부분
- 조합 해석을 4개에서 15개로 확장
- 조합 탐지는 강한 상태 라벨과 별개로 자기 역사 대비 상승·하락·보합을 판정
- stale 보조지표는 해석 지지·약화 집계에서 제외
- 같은 날짜에 여러 번 실행된 기록은 최근 30일 UI에서 마지막 스냅샷 1개만 표시
- Telegram 용어도 쉬운 표현으로 통일

## 화면 탭

1. **현재 상황** — 핵심 6개 최신값·상태·한줄 해석
2. **오늘의 해석** — 3축 조망·관찰 조합·보조지표 확인 결과
3. **최근 30일** — 지표별 흐름 + 8칸 상세 해석 카드
4. **같은 날짜 비교** — 공통 관측일 기준 비교
5. **상세 지표표** — 최신값과 변화·역사적 위치
6. **차트** — 원자료 흐름
7. **데이터 상태** — 운영 점검
8. **지표 설명** — 개별 지표 카드 + 지표를 같이 보는 법

## 사용자 용어

| 화면 | 내부 계산 |
|---|---|
| 약 1개월 변화 | 20개 관측치 전 대비 |
| 약 3개월 변화 | 60개 관측치 전 대비 |
| 최근 5년 위치 | 5년 point-in-time 백분위 |
| 최근 10년 위치 | 10년 point-in-time 백분위 |

내부 데이터 컬럼명은 하위호환을 위해 그대로 유지합니다.

## 오늘의 해석 원칙

- 하나의 조합에 정답 하나를 강요하지 않음
- 보조지표가 어떤 설명을 **지지하는지**와 **약화하는지** 분리
- 여러 설명을 동시에 지지하면 `복수 요인`으로 남김
- 데이터가 오래됐거나 판정불가이면 해당 확인을 보류
- `위험/안전/매수/매도` 같은 행동판단을 만들지 않음

## 데이터 파이프라인

```text
외부 스케줄러 / GitHub workflow_dispatch
  → run_refresh.py
  → FRED 핵심 6개 + 보조 3개 수집
  → 상태·3축·조합·조건부 해석 계산
  → HF Dataset versioned cache 저장
  → Telegram 알림

HF Space
  → 캐시만 읽어서 화면 표시
```

## 데이터 원칙

- raw 관측값만 계산 입력에 사용
- 상태는 매 refresh마다 raw에서 전체 재계산
- 백분위·변화·상태는 point-in-time
- Synced Snapshot은 forward fill 이전 실제 원본 관측일 교집합 사용
- 보조지표 실패가 핵심 6개 대시보드를 깨뜨리지 않음
- 같은 날짜 여러 스냅샷은 원본 버전에 보존하되 히스토리 UI는 마지막 한 개만 표시

## 핵심 원칙

> RiskRadar는 위험을 대신 판정하지 않습니다.
>
> 현재 어떤 지표와 축이 움직이는지, 어떤 설명이 가능하며 어떤 추가 데이터가 그 설명을 지지하거나 약화하는지 보여줍니다.
>
> 결과가 엇갈리면 억지로 하나의 결론을 만들지 않습니다.
